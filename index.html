INTRODUÇÃO AO HTML NA PRÁTICA

Os primeiros passos para aprender a criar e desenvolver uma página HTML é aprender a instalar e configurar o seu ambiente para desenvolvimento para então entender o funcionamento do inspetor de elementos do navegador. O uso de ferramentas para o desenvolvimento de páginas HTML não segue obrigatoriamente um editor de texto específico. No nosso caso indicaremos alguns programas e plugins que recomendamos.

FERRAMENTAS UTILIZADAS

Para desenvolvermos a criação da nossa primeira página usaremos o Visual Studio Code – VSC.

Se trata de um editor de código-fonte gratuito e multiplataforma, desenvolvido pela big tech Microsoft. Ele é uma ferramenta para desenvolvedores de software que permite criar e editar código em várias linguagens de programação.O programa pode ser baixado clicando no link abaixo:

    LINK para VSC

Siga as instruções contidas no próprio site para instalar o programa de acordo com as especificações de sua máquina. Uma dica que damos é instalar uma versão anterior a que o site indica pois será uma versão já testada e livre de possível erros de desenvolvimento que são regulares a uma nova atualização de qualquer programa.

Após instalado o programa em sua máquina, no ambiente do VSC, instale o plugin Live Server em sua aba de Extensões. Pode abrir a área usando o atalho Ctrl+Shift+X no seu teclado no VSC. Instalado o plugin, este te ajudará criando um servidor local para hospedar seu projeto e atualizar automaticamente uma página quando você fizer alterações no seu código. Feito isso poderemos passar para o próximo passo.

INSPETOR DE ELEMENTOS

O inspetor de elementos é uma ferramenta que nos permite visualizar e modificar o código fonte de uma página dentro do navegador. Para isso pode ser utilizado qualquer navegador que o usuário tenha o hábito de usar (Microsoft Edge, Google Chrome, Opera, Safari, FireFox, dentre outros). No nosso caso usaremos o Google Chrome. O inspetor de elementos pode ser aberto usando o atalho F12 do seu teclado ou no Personalizar e Controlar no Google Chrome (isso será em negrito). Ao abrir o inspetor, podemos observar vários elementos da construção da página tais como fonte, corpo do texto dentre outros que abordaremos logo mais.

ESTRUTURA BÁSICA DO HTML

Vamos criar nossa primeira HTML? Para isso no nosso VSC criamos uma pasta na qual iremos alocar nossos arquivos que iremos criar para a página. Nessa pasta criamos um arquivo, também pelo VSC com o nome de index.html, seguindo o sugerido pelas notações padrão de boas práticas de código.

O HTML consiste basicamente em 3 partes:

  1. Criamos uma Tag HTML usando o código tudo que ficar dentro desse código fará parte da página.
  2. Criamos uma nova Tag dentro da html que chamamos de Cabeça usando o código
  3. Dentro deste código criamos o Corpo da página com o código Tudo que estiver dentro desta tag irão aparecer para o usuário, ao contrário do Head que irá carregar antes de aparecer.

Estes são os elementos fundamentais usados para criar uma página. Tudo mais será incrementado a partir destas 3 tags. Por exemplo, dentro da Tag Cabeça podemos inserir a Tag e dentro dela podemos atribuir um título para nossa página que será observado na aba do navegador. Usando o Live Server podemos usar dentro do VSC o comando Alt+L ou Alt+O para observar tal feito. No nosso caso usaremos a tag Title Introdução ao HTML

FALANDO SOBRE TAGS

Como já mostramos, são 3 os tipos primordiais de tag: html, head e body.

Mas o que são elas? A tags (nome em inglês), são etiquetas que usamos nos códigos carregando funções dentro delas. Ou seja, as tags são usadas como identificação para modificações em nosso código. Muitas tags trazem consigo uma modificação ou aplicação a uma pagina que podem ser especificadas uma única vez como a tag (usada para imagens que abordaremos logo mais) ou precisam de comando de inicio e fim como a própria tag html que precisamos do comando marcando o início da página e marcando o fim. A barra então indicará o fim de uma tag. Mas quais são e como e quando usá-las?

As tags são muitas e cada uma carrega uma função e atributo ao código. Abordaremos agora algumas tags que são mais comumente utilizadas dentro das páginas e para que servem.

Itálico

Esta tag carrega a fonte itálica, uma tipografia que modifica a estrutura padrão deixando as letras levemente inclinadas. Servem para dar destaque ou enfatizar uma palavra, frase, sitação, etc dentro de um contexto no corpo do texto. Usualmente para palavras estrangeiras a linguagem do texto utilizada, para dar uma diferenciação em algo que queira exprimir dentro do texto, dentre outros.

Negrito

Esta tag deixa as letras do texto mais escuras e grossas, dando grande ênfase ao que se pretende dizer ou grande destaque em uma palavra. Usada quando quer se dar destaque ou deixar um forte contraste de algo em um texto.

Imput

Usado para atribuir um campo de entrada de texto ou números, de acordo com os atributos. Cria uma caixa de entrada na html onde pode ser inserido conteúdo pelo usuário.

Imagem

Insere uma imagem a html. Pode ser editada e configurada com o uso de atributos que farão a imagem ser inserida de acordo com a necessidade do desenvolvedor.

Lembrando que as tags podem ser usadas sozinhas ou separadamente, de acordo com a preferência ou intenção de que está escrevendo. Entretando, convencionalmente existem algumas orientações de boas práticas no uso das tags que para maior aprofundamento podem ser verificado no link a seguir:

ATRIBUTOS DAS TAGS

São propriedades que uma tag pode ter. Alguns atributos são mais genéricos, ou seja, podem ser usados em diversas tags. Já algumas são específicas a tag utilizada, atendendo os requisitos em particular daquele caso. Para isso listaremos alguns atributos utilizados em tags que são mais comuns de serem encontrados.

id=”titulo”

É um identificador de atribuição de título, deixando a fonte e o tamanho da fonte com as configurações que já vem pré-estabelecidas em Java Script pelo navegador.

style=”color: blue” ou class= “titulo-principal”

Mais utilizados no CSS, são comandos pré-definidos que orientam como o navegador irá formatar o texto tais como cor, tamanho da fonte, dentre outros.

img src=

O atributo "src=” possibilida incluir imagens no html que pode ser inserido a partir de um caminho que pode ser do uruário ou em link para página da internet.

Além disso podemos usar o atributo width="tamanho da imagem” atribuindo um número para tamanho da imagem.

TEXTOS

Existem tags específicas para textos, ou seja, tipografia html. Alguns com a finalidade bem específicas e mais usuais na criação de uma página.

Os Hs

As tags tipo H (h1, h2, h3, h4, h5 e h6), são atribuídas a títulos e subtítulos dentro de um texto dando uma formatação e um nível de acordo com cada desinência.

O h1 será o título principal de seu texto, sendo os demais subtítulos deste primeiro.

Parágrafo

A tag parágrafo é voltada para o elemento textual em si, ou seja, o que se pretende dizer na página, o conteúdo da informação precedida do título.

Citação

Quando houver necessidades de fazer uma citação durante o texto e deixar uma parte de destaque no seu html.

Exemplo:

"Você merece estar entre pessoas que não precise olhar no fundo delas para 'encontrar a parte boa'. Você merece receber oque tem de bom na superfície delas.”

LISTAS

As listas são utilizadas para pontuar, ordenado ou não, dentro do seu texto. De acordo com sua necessidade a lista pode ser ordenada por posição ou ícones onde não há critério de ordem.

Para listas ordenadas usamos o comando ol (ordinal list) e para listas não ordenadas usamos o comando ul (unordinal list).

Exemplo de lista ordenada no passo a passo de fazer um bolo:

  1. Pré aquecer o forno
  2. Bater a massa
  3. Passar a massa para uma forma untada
  4. Levar ao forno para assar.

Exemplo de lista não ordenada em uma lista de compras de supermercado:

  • arroz
  • açúcar
  • suco
  • detergente
